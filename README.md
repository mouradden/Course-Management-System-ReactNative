## Get started

1. for the front end, open a first terminal


   ```bash
   git clone git@github.com:mouradden/Course-Management-System-ReactNative.git
   ```

   ``` bash
   cd Course-Management-System-ReactNative
   ```

   ```bash
   npm i
   ```
2. for the back end, open a second terminal

   ```bash
    cd backend
   ```
   ```bash
    npm i
   ```
   ```bash
    npm start
   ```
